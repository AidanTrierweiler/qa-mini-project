## BUG-01: Incorrect error message displayed when login fields are empty

**Environment:**  
- Browser: Chrome  
- OS: Windows

**Steps to Reproduce:**
1. Navigate to the login page
2. Leave username field empty
3. Leave password field empty
4. Click Login

**Expected Result:**  
A validation message prompts the user to enter a username and password.

**Actual Result:**  
An error message is displayed stating "Your username is invalid!".

**Severity:**  
Low
